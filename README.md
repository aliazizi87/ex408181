# مثال برنامه‌نویسی شیءگرا در پی‌اچ‌پی: کلاس چکیده و ثابت کلاسی

این پروژه یک نمونهٔ آموزشی ساده از دو مفهوم پایه‌ای برنامه‌نویسی شیءگرا در زبان پی‌اچ‌پی است:

- **کلاس چکیده**: برای تعریف یک الگوی کلی که همهٔ کلاس‌های فرزند باید مطابق آن عمل کنند.  
- **ثابت کلاسی**: برای تعریف مقداری ثابت که در همهٔ کلاس‌های مرتبط به اشتراک گذاشته می‌شود.

---

## مفاهیم به کار رفته

### کلاس پایهٔ «شکل»
- این کلاس یک **کلاس چکیده** است و نمی‌توان از آن شیء ساخت.
- یک متد چکیده به نام «مساحت» دارد که هر کلاس فرزند باید آن را پیاده‌سازی کند.
- یک **ثابت کلاسی** به نام «واحد» تعریف شده که واحد نمایش مساحت را مشخص می‌کند:
